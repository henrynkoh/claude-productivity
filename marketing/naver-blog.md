# 네이버 블로그 포스트 (Naver Blog)

## 제목

시애틀 스타트업을 위한 ‘에이전트 스킬’ 1주 집중 프로그램 (컨텍스트 vs 슬롭)

## 본문

AI 모델 성능만으로는 부족합니다. **무엇을 매 턴마다 컨텍스트에 넣느냐**가 출력 품질을 가릅니다. 긴 `agent.md`류 파일은 토큰을 잡아먹고, 창이 채워질수록 모델이 “덜 똑똑해 보이는” 현상까지 만듭니다.

**Greater Seattle Agent Skills Week**는 시애틀 광역권의 **선발된 소수 스타트업 팀**을 대상으로, 7일 동안 다음을 실무로 체화합니다.

- **프로그레시브 디스클로저**: 스킬은 처음엔 **제목·한 줄 설명**만 노출하고, 작업이 맞을 때만 전체 지시를 로드합니다.
- **재귀적 스킬 빌딩**: 사람이 루프에 들어가 워크플로를 함께 실행하고, 실패 시 **증거(로그·재현)**로 원인을 좁힌 뒤, 성공한 실행을 스킬로 고정합니다.
- **다운로드 스킬 거절**: 마켓플레이스 범용 스킬보다 **우리 레포에서 성공한 실행**을 기준으로 작성합니다.
- **확장 절제**: 검증된 메인 에이전트 루프 전에는 서브 에이전트 남발을 하지 않습니다.

참고 영상(프로그램의 개념 축): [Greg Isenberg × Ross Mike — How AI agents & Claude skills work](https://www.youtube.com/watch?v=S_oN3vlzpMw)
