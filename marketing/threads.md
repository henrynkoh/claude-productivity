# Threads post pack

**Greater Seattle Agent Skills Week** — practical week for startup teams on agents, **context management**, and **skills** (not generic downloads). Reference interview: [YouTube](https://www.youtube.com/watch?v=S_oN3vlzpMw).

## Post 1

Most teams don’t need a novel-length AGENTS.md. They need **progressive disclosure**: short skill titles up front, full instructions only when the task matches. We teach that in one week—with daily artifacts.

## Post 2

**Recursive skill building:** run the workflow with the agent → debug with evidence when it fails → after a clean pass, freeze it as a skill. That’s the loop. Everything else is theater.

## Post 3

Scaling tip we hammer all week: **one strong main workflow first**. Add “marketing agent / biz agent” only when the parent path is measured and boring. Productivity > cool demos.
