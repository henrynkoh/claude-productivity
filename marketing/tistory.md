# 티스토리 포스트 (Tistory)

## 제목

[시애틀] 스타트업 팀용 ‘AI 에이전트·스킬’ 1주 커리큘럼 안내

## 본문

도구만 바꾸고 워크플로는 그대로면, 팀 생산성은 잘 오르지 않습니다. 특히 **항상 켜 두는 거대 시스템 프롬프트**는 비용과 품질 양쪽을 해칩니다.

이번 프로그램(**Greater Seattle Agent Skills Week**)은 **그레이터 시애틀** 지역의 선발 스타트업을 위해 설계되었고, 하루가 끝날 때마다 **실제 산출물**(규칙 정리, 스킬 인덱스, PR, 지표 등)을 남기는 것을 원칙으로 합니다.

핵심 흐름은 다음과 같습니다.

1. 글로벌 규칙 다이어트 — 불필요한 상식·중복 제거  
2. 스킬을 **카탈로그**처럼 설계 — 짧은 메타데이터 + 필요 시 전문 로드  
3. 성공한 실행을 스킬로 **동결** — 엣지 케이스마다 스킬 업데이트  
4. 한 가지 메인 루프가 단단해진 뒤에만 역할 분리 검토  

개념 배경으로 추천하는 영상: [How AI agents & Claude skills work](https://www.youtube.com/watch?v=S_oN3vlzpMw)
