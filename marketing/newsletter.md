Hello,

We’re opening a **small, selected cohort** for **Greater Seattle** startup teams that want **agent workflows** without the usual “AI slop.”

This program is built around ideas from a clear breakdown of **how agents and skills actually work**: **context management** beats raw capability, **skills** should use **progressive disclosure**, and the right way to grow capability is **recursive skill building**—teach the workflow, ship with evidence, then encode the win.

**What the week looks like (high level)**

- **Day 1–2:** Audit bloated global prompts; design a skill index (titles + descriptions first).
- **Day 3–4:** Run real workflows with the agent; debug with logs; capture success as versioned skills; avoid generic marketplace copy-paste.
- **Day 5–7:** Scale with discipline (one strong loop before specialists), capstone delivery, demo, and a **30-day adoption** plan.

If your team builds software and already experiments with tools like Claude Code or similar, this is meant to turn experiments into **repeatable operating standards**.
