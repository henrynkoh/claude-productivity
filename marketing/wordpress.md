## Title

Greater Seattle Agent Skills Week: A 7-Day Curriculum for Startup Teams (Agents, Context, Skills)

## Excerpt

A practical one-week program that helps Seattle-area startups adopt **agent workflows** without bloated global prompts—using **progressive disclosure**, **skills built from real runs**, and disciplined scaling.

## Full article

Startup teams do not need more generic AI tips. They need an operating system: **what** gets loaded into context, **when**, and **how** it is maintained when reality changes.

**Greater Seattle Agent Skills Week** translates that idea into daily execution. Participants work through:

- **Context as the moat** — audit and shrink always-on instruction files.
- **Skills as progressive disclosure** — design a skill index where only titles and short descriptions stay “visible” until a task triggers the full procedure.
- **Recursive skill building** — pair with the agent on a real workflow, debug with evidence, then capture the clean pass as an updated skill.
- **Build, don’t bulk-download** — reject generic marketplace skills that ignore your data boundaries and brand voice.
- **Scale with discipline** — strengthen one main agent loop before adding specialized sub-agents.
- **Capstone and demo** — ship evidence-backed outcomes and a 30-day adoption plan.

The curriculum is aligned with a widely shared interview on agents and skills: [How AI agents & Claude skills work (Clearly Explained)](https://www.youtube.com/watch?v=S_oN3vlzpMw).
